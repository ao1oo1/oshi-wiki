# 認証セキュリティ対策

## 不審IPアドレスの遮断

`.env`へカンマ区切りで設定する。

```env
SECURITY_BLOCKED_IPS=203.0.113.10,198.51.100.0/24
```

単一IP・CIDRの両方に対応する。

対象：

- 一般・Writer・管理者ログイン
- ユーザー登録
- パスワード再設定
- メール認証
- パスワード確認・変更
- プロフィール変更
- アカウント削除
- スタッフプロフィール変更

## ログイン失敗制限

標準値：

- メールアドレス＋IP：15分間に5回失敗で一時制限
- IP全体：15分間に20回失敗で一時制限
- ログイン成功時に該当カウントを解除

環境変数：

```env
SECURITY_LOGIN_ACCOUNT_MAX_ATTEMPTS=5
SECURITY_LOGIN_IP_MAX_ATTEMPTS=20
SECURITY_LOGIN_DECAY_SECONDS=900
```

## その他の回数制限

- 新規登録：10分間に5回
- パスワード再設定メール：10分間に5回
- パスワード再設定：10分間に5回
- パスワード変更：10分間に5回
- プロフィール更新：10分間に10回
- アカウント削除：10分間に5回
