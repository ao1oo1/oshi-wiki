# 最高管理者 Writerアナリティクス

## 目的

最高管理者がWriter会員数、Oshi-Wiki Plus契約状況、Writerデータ利用状況、
課金状態の不整合を個人の創作本文を閲覧せずに把握できるようにする。

## URL

- 画面: `/admin/analytics`
- CSV: `/admin/analytics/export`

## 権限

- 最高管理者のみ
- スタッフは403
- 個々のストーリー本文、プロンプト本文、キャラクター設定本文は表示しない

## Phase 1

- Writer総登録者数・有効会員数・期間内新規登録者数
- Oshi-Wiki Plus会員数・加入率・解約予約・支払い猶予
- Writer登録者数の日別推移
- Plus契約プロフィール作成数の日別推移
- キャラクター、関係性、保存プロンプト、ストーリー、文体分析の件数
- データ構成比、文字量、推定保存量
- 無料会員とPlus会員の平均登録件数比較
- 無料会員の上限80%以上・上限到達者数
- Stripe契約とOshi-Wikiプランの不整合
- Webhook処理失敗
- 集計CSV出力

## 集計上の注意

Plus推移は、現在保存されている課金プロフィールの作成日を基準とする。
正確な加入・解約履歴、DAU、MAU、継続率、料金ページから決済までのファネルは、
Phase 2の利用イベント記録導入後に集計する。
