# Oshi-Wiki v2：AI執筆補助機能

## v2の目的

v2では、AI本文生成は実装せず、AIに貼り付けるためのプロンプト作成・保存・コピー機能を実装する。

## v2で実装すること

- 一般ユーザー登録
- 一般ユーザーログイン
- 一般ユーザー用マイページ
- オリジナルキャラクター登録
- オリジナルキャラクター関係性登録
- v1の作品・キャラクター・関係性データ参照
- v1データとv2データを組み合わせたプロンプト作成
- プロンプト保存
- プロンプトコピー

## v2では実装しないこと

- OpenAI API連携
- Ollama連携
- サイト内AI本文生成
- AIによる自動執筆

## DB使用制限

1ユーザーあたり：

- オリジナルキャラクター：最大30件
- 関係性：最大100件
- 保存プロンプト：最大50件
- 1プロンプト本文：最大20,000文字
- あらすじ：最大5,000文字
- 各備考：最大2,000〜5,000文字

## 実装フェーズ

1. v2ブランチ作成・v1タグ確認
2. 一般ユーザー登録/ログイン/マイページ
3. DB使用制限の共通定数作成
4. オリジナルキャラクターDB/CRUD
5. 関係性DB/CRUD
6. プロンプト作成DB/CRUD
7. v1データ抽出
8. プロンプト生成ロジック
9. コピー機能・UI調整・v2タグ保存

### 最高管理者の制限

v2のDB使用制限は一般ユーザー向けの制限とする。

最高管理者権限保持者は、以下の制限を受けない。

- オリジナルキャラクター登録数
- 関係性登録数
- 保存プロンプト数
- プロンプト本文文字数
- あらすじ文字数
- 備考文字数

実装上は、`App\Support\WritingAssistLimits` でユーザーを受け取り、最高管理者の場合は `null` を返す。  
`null` は「制限なし」として扱う。
